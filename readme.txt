== Description ==

Display a notification bar to show how your site complies with the UK & EU Cookie Law. Custom design settings are available.

== Installation ==

1. Extract and Upload 'google-plugin.zip' into your wp-plugins directory
1. Activate the plugin from the admin panel of WP
1. Visit your website and the notification bar should be displayed.
